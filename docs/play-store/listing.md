## 7. Hi-res Icon(必填,512×512 PNG,32-bit)

跟 launcher icon 不同檔!Play 列表用。

### 怎麼產

1. 我已經把 launcher 的 adaptive icon 改成放大鏡 vector
2. 在 Android Studio 開檔 → File → New → Image Asset → Launcher Icons (Adaptive and Legacy)
3. 在精靈裡選 Foreground = `ic_launcher_foreground`(我們新的)、Background = `ic_launcher_background`(我們新的)
4. 預覽 OK 後 Finish — 會重新產所有 mipmap webp + 給你預覽圖
5. 從預覽匯出 512×512 PNG 上傳到 Play

替代法(不開 Android Studio):
- 用 [Android Asset Studio Launcher Icon Generator](https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html) 上傳 SVG 線上產
- 或用 [IconKitchen](https://icon.kitchen)

- **Category**:Tools(工具) — 主要

## 10. 上架後可立即做的 ASO 持續優化

1. 上線後 1-2 週看 Play Console「使用者擷取分析」哪個關鍵字帶來最多安裝
2. 把高效關鍵字往標題 / 短描述前面挪
3. A/B 測試 feature graphic(Play Console 內建 Store Listing Experiment)
4. 收 GitHub Issue 高頻請求 → 下版本更新時把對應 feature 寫進 What's new 提高更新點擊
